List of Housing developments: feature.

Who is the list of housing developments for? What does it need to do? What have we thought about? What are we doing now?

Who is the list of housing developments for?

There is a need for anyone to be able to find a housing development, and to be able to see what is being developed. Transparency.

There is no specific need from a particular specialist user group to find a housing development, in the way we currently show. We expect specialists and people buying homes to have the URL to a more specific page - the development or property they are interested in. To have been given it, or to have found that via google.

Once information is published on GOV.UK about housing development content, homes under development, and transactional services and information about developing, buying and selling, there is a need to be able to find developments and navigate between general content and specific content

In the short term, it acts as a landing page for the service

What does it need to do?

  • Understand what this page contains
  • where they might want to be instead
  • where they want to go next (what do they want to do next?)
  • quickly identify the housing development they are interested in

What have we thought about?

  • Where would this sit on gov.uk - what would link into it?
  • what would people be searching for to find this page (key words)
  • are developments known by more than one name (the name changes over time?)
  • the address changes over time
  • how might people want to search for a development in the collection - name, type, geography, by location, size, last updated, when initially published, how complete (how close to finishing)
  • searching, filtering, ordering, pagination, ordering
  • the list is 'active developments' - what happens when the development is finished? (is there an historical list?)
  • publishing a data version - json or csv (or API) - for lenders? for anyone?

What are we doing now?

Set the context : say what the list is, what we are doing, how to use 'EPA' / housing services.

list all developments, on one page - in alphabetical order. So they are easy to find.
